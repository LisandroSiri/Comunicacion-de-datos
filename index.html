Proyecto Final: Monitoreo en Tiempo Real

Sistema IoT de Monitoreo de Sensores

Sistema que permite monitorear en tiempo real los valores de temperatura, humedad, concentración de gases y distancia, utilizando un microcontrolador Arduino UNO con Shield Ethernet.

Temperatura

Sensor DHT11 con rango de 0°C a 50°C

Humedad

Medición de humedad relativa 0-100%

Gases

Sensor MQ-2 para calidad del aire

Distancia

Sensor ultrasónico HC-SR04

Visualizacion

LCD 20x4 para Visualizacion local
